The Problem

Over 45% of all web traffic is made by bots — not people. AI models and scrapers crawl millions of websites daily, copying data, training algorithms, and draining server resources without permission.

As WIRED reported: “Web publishers are being quietly stripped of their traffic while their content trains AI systems for free.”

This leads to lower ad revenue, slower sites, and fewer page views for real users. Many bots ignore robots.txt and spoof browser headers to get in unnoticed.

Our Solution

GhostLink is an easy-to-deploy tracker that logs every visit to your site. It identifies bot traffic, logs it with full metadata, and blocks known bad actors in real time — helping you regain control over who uses your data.

  • Detects bots using user-agent and header patterns
  • Blocks known scrapers and loggers
  • Logs IP, user agent, timestamp, and detection flag
  • Simple web UI to view logs and export data

How to Install GhostLink (No Coding Required)

  1. Download the GhostLink ZIP file from the button above.
  2. Unzip it — inside, you’ll see a file called ghost.js.
  3. Upload ghost.js to your website's file manager (for example, into a /static or /js folder).
  4. Edit your homepage and add the following line inside the <head> tag:

    <script src="/path/to/ghost.js"></script>

    Replace /path/to/ghost.js with the correct location (like /static/ghost.js).
  5. Save and upload your homepage if needed.
  6. That’s it — GhostLink is now tracking visits! You can view logs on our dashboard.
